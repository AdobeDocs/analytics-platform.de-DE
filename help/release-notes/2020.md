Januar 2021 | Journey IQ: Die Cross-Channel Analytics ermöglicht es Kunden, einen Adobe Analytics-Ereignis-Datensatz (oder einen anderen) im Adobe Experience Platform Data Lake von einem ID-Namespace zum anderen erneut zu verschlüsseln. Dies bedeutet in der Regel, dass der Ereignisdatensatz von einer Cookie-basierten ID neu in eine personenbasierte ID verschlüsselt wird.
