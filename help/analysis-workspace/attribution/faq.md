# Häufig gestellte Fragen

Im Folgenden finden Sie Antworten auf häufig gestellte Fragen zur Attribution.

+++Was bedeutet der *Keine* bei Verwendung von Attribution?

Der *Keine*-Zeileneintrag ist ein Sammelbegriff für alle Konversionen, die ohne Touchpoints im Lookback-Fenster durchgeführt wurden. Um die Anzahl der Konversionen zu reduzieren, die dem Zeileneintrag *Keine* zugeordnet sind, versuchen Sie es mit einem längeren Lookback-Fenster.

+++


+++Sind Attributionsmodelle in anderen Analytics-Funktionen verfügbar?

Ja. Attributionsmodelle sind auch als Teil der Komponenteneinstellungen für eine Metrikkomponente in einer Datenansicht verfügbar. Siehe [Attribution - Komponenteneinstellungen](/help/data-views/component-settings/attribution.md). Für Dimensionskomponenten können Sie Persistenz verwenden. Persistenz ist die Fähigkeit, dass ein bestimmter Dimensionswert einer Metrik über das Ereignis hinaus zugeordnet werden kann, für das er festgelegt ist. Dafür wird eine Kombination aus Zuordnung und Gültigkeit verwendet. Siehe [Persistenz - Komponenteneinstellungen](/help/data-views/component-settings/persistence.md) für weitere Informationen

+++


+++Welche Dimensionen und Metriken werden unterstützt?

Die Attribution wird für alle Nicht-Standard-Metriken unterstützt, die Sie in Ihrer Datenansicht definiert haben.

+++



+++Funktioniert die Attribution mit Suchdaten?

Ja, Dimensionen aus Suchdaten werden vollständig unterstützt.
